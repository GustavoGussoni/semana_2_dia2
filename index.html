    Camadas Box Model
  • conteúdo - content (é o valor do elemento e o que está contido dentro das tags)
  • preenchimento - padding (topo, direita, baixo, esquerda) (é o valor de preenchimento. O padding é a distância do conteúdo até a borda)
  • borda - border (é o limite que separa o preenchimento da margem. Faz a função de uma borda mesmo.
  • margem - margin (é o valor externo a borda do elemento. è responsável pela separação entre um elemento e outro)

    • DICA! O box-sizing propriedade CSS nos permite alterar as regras para cálculos de tamanho. O valor padrão ( content-box) só leva o conteúdo interno em conta, mas oferece um valor alternativo: border-box. Com box-sizing: border-box, as coisas se comportam de maneira muito mais intuitiva.

    Unidades de medida do css
  • Unidades de medida absolutas: são definidas pela física, como o píxel, centímetro, metro, etc. são medidas fixas e não mudam conforme as especificações do dispositivo. são unidades estáticas.
    1. pixel - é um ponto na tela. como ele é uma unidade de medida absoluta, seu valor é fixo. um ponto da tela é um ponto na tela, independente da tela.
  • Unidades de medida relativas: são calculadas tendo como base outro elemento ou unidade de medida definida. essas unidades são ajustadas conforme o comportamento da tela e com base nas definições realizadas são unidades flexíveis
    1. rem - é uma unidade de medida relativa, ou seja, vai depender de outro elemento. o rem é relativo ao font size definido no root.
    2. em - é uma unidade de medida relativa,, ou seja, vai depender de outro elemento. o em é relativo ao font size definido no elemento pai.
    3. percentagem - é uma unidade de medida relativa, ou seja, vai depender de outro elemento, ou unidade de medida. 0% é definido relativo ao tamanho do pai do elemento.
    4. vw - é uma unidade de medida relativa, ou seja, vai depender de outro elemento, ou unidade de medida. o vw é definido relativo ao tamanho da view, janela. o vw define o comprimento do elemento e cada unidade corresponde a 1% da janela.
    5. vh - é uma unidade de medida relativa, ou seja, vai depender de outro elemento, ou unidade de medida. o vh é definido relativo ao tamanho da view, janela. o vh define a altura do elemento e cada unidade corresponde a 1% da janela.
Aula 12

  • % - relativo ao valor do elemento pai dessa propriedade
  • em - relativo ao tamanho da fonte atual do elemento
  • rem - relativo ao tamanho da fonte do root da página (por exemplo, o elemento html). "rem" = root)
  • vh - relativo à altura da janela de visualização (desktop ou mobile). 1 vh = 1/100 da altura da janela de visualização
  • vw - relativo à largura da janela de visualização. 1vw = 1/100 da largura da janela de visualização
  • vmin - relativo à menor dimensão da janela de visualização (por exemplo, para orientação de retrato, a largura é menor do que a altura, portanto, é relativa à largura). 1vmin = 1/100 da dimensaão menor da janela de visualização
  • vmax - relativo à maior dimensão da janela de visualização (por exemplo, altura para orientação de retrato). 1vmax = 1/100 da dimensão maior da janela de visualização.

Nem sempre está claro qual dessas opções é melhor usar para cada tipo de propriedade CSS. Por exemplo, % é geralmente mais apropriado para propriedades relacionadas ao layout, como largura, do que para tamanho da fonte.
  • Percentual(%): Você deseja que um elemento filho tenha 10% da largura do pai como margem para que nunca preencha todo o elemento pai. Se o tamanho do pai mudar, a margem também será atualizada.
  • em: Você deseja que a fonte de um elemento filho tenha metade do tamanho da fonte de seu pai(por exemplo, o parágrafo sob o título de uma seção).
  • rem: O tamanho da fonte deve ser o dobro do tamanho da fonte do elemento raiz. Pode ser assim que você dimensiona seus cabeçalhos, porque todos eles devem ter o mesmo tamanho, independentemente do contêiner pai.
  • vh: Sua página deve sempre ter a altura da janela de visualização.
  • vw: Você tem uma seção com texto que deve ter a metade da largura da janela de visualização.
  • vmin: Você tem uma imagem que deve ser sempre tão larga quanto a menor dimensão da janela de visualização. Em um dispositivo móvel no modo retrato, a imagem será tão ampla quanto a largura da janela de visualização.
  • vmax: Você não se importa se uma imagem for cortada porque você deseja que ela preencha completamente a dimensão maior da janela de visualização. Por exemplo, se a imagem de um padrão for usada como plano de fundo.
      • Dica!
    • No geral, quando e como você escolhe suas unidades CSS se resumem a algumas perguntas:
    • Eu quero que o meu estilo seja dimensionado quando o tamanho da janela de visualização mudar?
    • Se eu quiser que ele seja escalonado, ao que eu quero que ele seja relativo? Depois de responder a essas perguntas, é muito mais fácil definir qual unidade usar!
Pseudo-Classe - Aula 14
    É uma palavra-chave adicionada a seletores do CSS que específica um estado especial do elemento selecionado.
      Principais Pseudo-Classes
    1. :hover - Usado para atribuir um efeito ao elemento selecionado, quando o usuário mover o mouse sobre a àrea do elemento.
        exemplo: .box { width: 100px; height: 100px; background-color: blue; } .box:hover { background-color: green; }
        nesse caso, quando se passa o mouse em cima do quadro azul, ele fica verde.
    2. :focus - Usado para estilizar um elemento que está em foco, que pode acontecer quando o usuário usa o teclado para selecionar, com a tecla TAB por exemplo, ou quando seleciona com o mouse em um campo de formulário.
        exemplo: .input { height: 30px; padding-left: 20px; border: 2px solid transparent; } .input:focus { border-color: green; } .button { height: 30px; width: 70px; margin-left: 10px; border: 2px solid transparent; } .button:focus { border-color: green; }
    3. :nth-child() - Usado para selecionar elementos com base em suas posições em um grupo de elementos irmãos, recebendo um padrão como parâmetro para poder selecionar os elementos.
    4. valores
        odd - representa elementos cuja posição numérica em uma série de irmãos seja ímpar: 1, 3, 5, etc.
        even - representa ele,entos cuja posição numérica em uma série de irmãos seja par: 2, 4, 6, etc.
    5. notação funcional
        An+B Representa elementos cuja posição numérica em uma série de irmãos corresponda ao padrão An+B, o qual será aplicado a todo número maior ou igual a zero (ou seja, começando do zero). O índice do primeiro elemento é 1. Ambos os valores A e B devem ser integer (en-US) (inteiros).
      exemplos
    1. tr:nth-child(odd) ou tr:nth-child(2n+1) Representa as linhas ímpares de uma tabela HTML: 1, 3, 5, etc.
    2. tr:nth-child(even) or tr:nth-child(2n) Representa as linhas pares de uma tabela HTML: 2, 4, 6, etc.
    3. :nth-child(7) Representa o sétimo elemento.
    4. :nth-child(5n) Representa os elementos de número 5 [=5vezes1], 10 [=5vezes2], 15 [=5vezes3], etc.
    5. :nth-child(3n+4) Representa os elementos dvezes número 4 [=(3vezes0)+4], 7 [=(3vezes1)+4], 10 [=(3vezes2)+4], 13 [=(3vezes3)+4], etc.
    6. :nth-child(-n+3) Representa os primeiros três elementos. [=-0+3, -1+3, -2+3]
    7. p:nth-child(n) Representa todos os elementos p em um grupo de irmãos. Isso seleciona os mesmos elementos que um simples seletor p faria (só que com um nível maior de especificidade).
    8. p:nth-child(1) or p:nth-child(0n+1) Representa todo p que seja o primeiro de seu grupo de irmãos. Ele é idêntico ao seletor :first-child (e tem o mesmo nível de especificidade).
    9. p:nth-child(n+8):nth-child(-n+15) Representa do 8º até o 15º elementos p de um grupo de irmãos.
